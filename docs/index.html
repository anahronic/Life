Ayalon Real-Time Traffic Monitor

This repository contains a Python/Streamlit app (traffic_app.py) that requires a server runtime and a secret TOMTOM_API_KEY. GitHub Pages can host only static files, so the live dashboard must be hosted elsewhere.

Next step: deploy the Streamlit app on Streamlit Community Cloud and paste the URL below.

Deploy on Streamlit Cloud View repo on GitHub

Local run

Run locally (WSL/Linux/macOS):

pip install -r requirements.txt
streamlit run traffic_app.py

Tip: create .streamlit/secrets.toml and set TOMTOM_API_KEY there.
